# 로또 번호 추첨 (Lotto Draw)

1~45 중 무작위 6개 + 보너스 1개를 추첨하는 웹 서비스입니다.  
토스 스타일 UI와 실제 추첨처럼 번호가 하나씩 공개되는 연출을 적용했습니다.

## 실행 방법

- `index.html` 파일을 브라우저에서 열기
- 또는 로컬 서버 실행: `npx serve .` 후 접속

## Git 저장소 연결 및 푸시

이 프로젝트를 GitHub 저장소에 올리거나 업데이트할 때 아래 순서대로 진행하세요.

### 최초 1회 (저장소 연결)

```bash
cd c:\Users\SD2-02\Desktop\260311
git init
git remote add origin https://github.com/dominosmkt777/20260311.git
```

### 업데이트할 때마다 (커밋 & 푸시)

```bash
git add .
git commit -m "로또 추첨 페이지 업데이트"
git branch -M main
git push -u origin main
```

이미 `origin`이 있다면:

```bash
git add .
git commit -m "업데이트 내용 요약"
git push origin main
```
